# MIRA Marketing Video Scripts — for Manufacturing

Scene-by-scene generation scripts for **AI video models** (MiniMax **Hailuo** / Google **Veo 3**).
Each scene = **one 8-second clip**. Generate every scene separately, then **merge in order** into a single video (CapCut / Premiere / DaVinci Resolve).

Each video file (`video1.md`, `video2.md`, `video3.md`) contains **4 scenes** → a finished **~32-second** ad.

## How to use each scene block

Every scene gives you five things:

1. **🎬 VISUAL PROMPT** — paste this straight into Hailuo/Veo. It fully describes subject, setting, camera, motion, lighting, style, and aspect ratio.
2. **🎥 CAMERA / MOTION** — the movement to keep (repeat it in the prompt if the model drifts).
3. **🎙️ VOICEOVER (VO)** — the narration line. ~18–20 words = ~8 seconds at a calm pace.
4. **🔤 ON-SCREEN TEXT** — add this in your editor **in post** (AI models render text unreliably — do NOT rely on the model to spell it).
5. **🎵 AUDIO / SFX** — music mood + sound effects. On **Veo 3** you can generate native audio; on **Hailuo** add audio in post.

## STYLE BIBLE — keep these identical across every scene so clips merge seamlessly

- **Aspect ratio:** 16:9, cinematic, 24fps filmic look, photorealistic, shallow depth of field.
- **Grade / palette:** cool graphite industrial base (#1F2937), MIRA accent = **electric blue #3B82F6** + **soft purple #8B5CF6** glow; success moments flash **green #22C55E**. Warning moments use a controlled amber/red.
- **How MIRA appears on screen:** NEVER a humanoid robot or a glowing brain. MIRA is an **ambient intelligence** — a calm teal-blue voice, a soft pulse of light, and clean holographic UI cards / control-room screens. Think premium Apple-style product film, not sci-fi.
- **Recurring human:** **Arjun**, 40s, plant maintenance manager, short dark hair, light stubble, navy polo + hi-vis safety vest, calm and competent. Keep his look identical in every scene.
- **Setting:** modern, clean manufacturing plant — CNC machines, robotic arms, conveyor lines, and a glass-walled control room with large wall dashboards.
- **Tone:** calm, premium, confident. Never hype, never cartoonish.
- **Global style tag to append to every prompt:** `cinematic, photorealistic, volumetric light, smooth motion, shallow depth of field, high detail, 16:9, no on-screen text, no captions`.

### Brand rules (from the MIRA brand guide — follow strictly)
- Refer to MIRA as **"she" / "your Digital Facility Manager."** Never "chatbot," "AI agent," or "bot."
- **Outcome first**, not features. Calm executive confidence.
- No stock robots, brains, or glowing-network clichés.

## Merge & post-production checklist

1. Generate all 4 scenes per file (regenerate any clip whose motion/continuity drifts).
2. Sequence them 1 → 2 → 3 → 4. Use a fast **0.2–0.3s cross-dissolve** or hard cut between scenes.
3. Add the **VO** (record a calm human voice, or a premium TTS) on one track; duck the music under it.
4. Add **on-screen text** and the **MIRA wordmark + tagline** as overlays (models can't spell reliably).
5. End card: MIRA glyph + **"Your Digital Facility Manager."** + CTA (website / "Book a 30-min walkthrough").
6. Export 1080p (or 4K) 16:9. Also crop a 9:16 vertical version for Reels/Shorts if needed.
